# nodejs-tcp-client-server
Simple tcp client and server in Node.JS


## Run Quick demo on your local machine

1. **Clone repo**
```
git clone https://github.com/sysleaf/nodejs-tcp-client-server.git
cd nodejs-tcp-client-server
```

2. **Start TCP Server**
```
node server.js
```

3. **Start TCP Client**
```
node client.js
```
